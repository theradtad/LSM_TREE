# LSM Tree Implementation

A Java implementation of a Log-Structured Merge Tree (LSM Tree).

## Requirements

- JDK 17 or higher
- Maven 3.6 or higher

## Building the Project

To build the project, run:

```bash
mvn clean install
```

## Running Tests

To run the tests:

```bash
mvn test
```

## Project Structure

- `src/main/java` - Main source code
- `src/test/java` - Test source code
- `pom.xml` - Maven project configuration
